With so many different Metro templates in the market, We try to build it with the best template support. It can be difficult to know the best to use. But we try to make it easy to build best template for everyone.


## Features ##

 - jQuery 2.0.2
 - jQuery UI 1.10.3
 - Bootstrap 2.3.2
 - jQuery UI Touch Punch
 - Responsive
 - Documentation
 - Available 20+ beautiful colors for each component
 - Custom bootstrap component
 - Ajax Loader
 - Alert & Notification (bootbox & pnotify)
 - Gauge
 - FullCalendar
 - Sliders & Bars
 - Tiles
 - Looper
 - Splash page
 - Appbar
 - Custom Form Element (input, checkbox, radio, buttons)
 - Select2
 - ColorPicker
 - DatePicker

 - DateTimePicker
